== Description ==

WooCommerce conditional blocks allows you to display additional or alternate content based on a set of rules and conditions, including current users role, product categories, product tags, prices, cart contents, and many, many more.

== Installation ==
* Upload the `woocommerce-conditional-blocks-x.x.x.zip` file through the WordPress Plugin screen and install
* Otherwise please upload the uncompressed `woocommerce-conditional-blocks` folder to the `wp-content/plugins/` directory
* Activate the plugin from the Plugins page within WordPress
